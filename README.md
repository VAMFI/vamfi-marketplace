# VAMFI Plugin Marketplace

Claude Code plugins from VAMFI.

## Install

```bash
/plugin marketplace add VAMFI/vamfi-marketplace
/plugin install skill-forge@vamfi-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [skill-forge](https://github.com/VAMFI/skill-forge) | Create research-grounded Claude Code skills with verified sources |

## Add a Plugin

Edit `.claude-plugin/marketplace.json` and submit a PR.
